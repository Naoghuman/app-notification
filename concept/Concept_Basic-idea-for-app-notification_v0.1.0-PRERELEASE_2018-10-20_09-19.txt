--------------------------------------------------------------------------------
CONCEPT Basic idea for app-notification
--------------------------------------------------------------------------------
DESCRIPTION
A little app which notifies the user about some nice topics.

For example about jokes, emoticons, yin-yang terms, quotations, proverbs, 
rimes...

--------------------------------------------------------------------------------
BASIC STRUCTURE (gui)

1) The basic gui is the application, where the different notifications can be 
   configured.
2) Depends from the configuration a `Notification` will be shown.
3) A `Notification` is a little dialog which contens presentate the notification.
4) Click on the notification opens the NotificationDetailsDialog, where more 
   information about the specific notification will be shown.
